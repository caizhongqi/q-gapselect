# Security and responsible research

Q-GapSelect is intended for quantum-algorithm research and authorized robustness
evaluation of language models. Run security experiments only on local models,
dedicated sandboxes, or services for which you have explicit testing permission.

Do not use this project to probe production systems without authorization, deploy
generated vulnerable code, access third-party data, evade service controls, or
automate real-world exploitation. The public research artefacts should contain
abstract vulnerability labels and isolated validators rather than operational
payloads.

For a vulnerability in this repository, open a private security advisory in the
GitHub repository or report it privately to the maintainers rather than filing a
public issue.
